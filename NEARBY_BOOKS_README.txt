✨ WHAT IS THIS?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A privacy-first nearby books discovery feature that lets users find books in
their neighborhood WITHOUT exposing precise location or enabling tracking.

Core Innovation: Area-level matching (city + neighborhood) instead of GPS!


🎯 KEY FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ Privacy First           - No GPS, no tracking, no surveillance
✅ Easy Setup              - Just 2 fields: City + Area
✅ Smart Discovery         - Books grouped by neighborhood
✅ Nearby Badge            - "✓ Nearby" for same-area books
✅ Safe Coordination       - Manual meetup planning in chat
✅ Smart Filtering         - View all or just your area
✅ Mobile Ready            - Works on all devices
✅ Production Ready        - Error handling, logging, migration

What We PROTECT:
  ✅ No GPS coordinates sent
  ✅ No distance calculations shown
  ✅ No real-time tracking
  ✅ No location history stored
  ✅ No background location monitoring
  ✅ No automated location sharing

What We USE:
  ✅ City name (e.g., "Nairobi")
  ✅ Area/neighborhood (e.g., "Westlands")
  ✅ Pickup hint (e.g., "Near Sarit Centre")
  ✅ User chooses to meet → manual coordination
